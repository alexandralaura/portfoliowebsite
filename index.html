Piano Keys

The aim of this project is to create an interactive game to help a beginner level piano student. I created a piano player with DOM events in JavaScript. I created a function that changes the background color of the keys when they are pushed down.
function keyPlay(Event) {
Event.Target.style.backgroundColor = 'red'; };
I created a function named keyReturn that returns the background colour of the keys to their default when the mouse is released.
function keyReturn(Event) {
Event.Target.style.backgroundColor = ''; };
These functions are now used as event handler properties.
function functionname(note) {
note.addEventListener('mousedown', keyPlay);
note.addEventListener('mouseup', keyReturn); }; Now I create a function that passes my notes array through my event handler function.
notes.forEach(functionname);
At this stage the program knows what to do when each key has a mousedown or mouseup event fired on it.
